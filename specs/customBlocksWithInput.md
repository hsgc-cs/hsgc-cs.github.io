# Custom Blocks With Input

Please create a project and write all of these custom blocks. Hint: Start at the top.

## 1 input

* triple
* absolute value
* increment

## 2 inputs

* sum
* difference
* product
* quotient
* x to the power of y (exponent)
* maximum
* minimum

## ‘list’ input

* sum
* product
* maximum
* minimum
* does list contain x?
* index of x?
* MapReduce
